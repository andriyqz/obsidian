### Компроміси при позначенні відсутніх даних:
Ми повинні пояснити комп'ютеру, як нам позначити відсутність даних в якійсь комірці. Для цього є методи, одними з яких я маскування та значення-індикатори.

Маскування - це коли ми створюємо новий масив зі значень True, False який буде відповідати за те, чи відповідне значення за індексом присутнє в датасеті. Мінусом цього підходу є те, що нам знадобиться в два рази більше об'єму пам'яті та часу, для того щоб проводити перевірку на присутність.

Використання значень-індикаторів полягає в тому, щоб ввести якесь зарезервоване значення, яке буде означати, що комірка пуста: наприклад, -1999, рядок "None".
Проте у цього підходу є мінус: у нас зменшується діапазон значень, які ми можемо покласти в цю комірку.

### Відсутні значення в бібліотеці Pandas:
Pandas використовує None із Python та np.nan (dtype=float) із NumPy.
np.nan - це число з плаваючою точкою, зарезервоване IEEE., integer аналога немає, тому якщо np.nan в int серії pandas, то бібліотека конвертує його у формат числа з плаваючою точкою.

```python
data = pd.Series([1, np.nan, 'hello', None])

print(data.isnull())
print(data[data.notnull()])
print(data.dropna())
```

### Заповнення пропущених значень
```python
df = pd.DataFrame([[1, np.nan, 2],
                   [2, 3, 5],
                   [np.nan, 4, 6]])

print(df.dropna())
print(df.dropna(axis='columns'))
df.iloc[2, [1, 2]] = None
print(df.dropna(how='all'))
print(df.dropna(axis='rows', thresh=3)) # thresh = 3 це кількість непустих значень в рядку, щоб він не був видалений
```

```python
data = pd.Series([1, np.nan, 2, None, 3], index=list('abcde'))

print(data.fillna(0))
print(data.ffill()) # заповнення кроком вперед
print(data.bfill()) # заповнення кроком назад
```
