### Series:
Об'єкти Series своєю природою нагадують комбінацію словника та ndarray

```python
data = pd.Series([0.25, 0.5, 0.75, 1.0])

print(data.values)
print(data.index) # RangeIndex(start=0, stop=4, step=1)
print(data[1])
print(data[1:3])

data
```

Індексація задається явно:
```python
data = pd.Series([0.25, 0.5, 0.75, 1.0],
                index=['a', 'b', 1, 'd'])
print(data['b'])

data
```

Series можна задати і словником, для Series є різні фішки індексації, як в numpy масивах, тільки вони працюють з ключами більшості типів даних.
```python
population_dict = {'California': 38332521,
                   'Texas': 26448193,
                   'New York': 19651127,
                   'Florida': 1325345,
                   'Illinois': 2354123
                   }

data = pd.Series(data=population_dict)
print(data['Texas':'Florida'])
```

```python
data = pd.Series(5, index=[100, 200, 300])
data
# 100    5
# 200    5
# 300    5
# dtype: int64
```

Pandas не сортує ключі сам:
```python
data = pd.Series({2: 'a', 1: 'b', 3: 'c'})
data.sort_index()
```

Зберігаються тільки значення за індексом 3 і 2:
```python
data = pd.Series({2: 'a', 1: 'b', 3: 'c'}, index=[3, 2])
data
```

### DataFrame:

Об'єкт DataFrame є аналогом двовимірного масиву ndarray з додатковим функціоналом, також його можна сприймати як масив об'єктів Series:
```python
area_dict = {'California': 1, 'Texas': 2, 'New York': 40,
             'Florida': 35, 'Illinois': 14}

population_dict = {'California': 38332521,
                   'Texas': 26448193,
                   'New York': 19651127,
                   'Florida': 1325345,
                   'Illinois': 2354123
                   }

area = pd.Series(area_dict)

population = pd.Series(population_dict)
states = pd.DataFrame({'population': population, 'area': area})

print(states.columns)

states
```

DataFrame з однієї колонки Series:
```python
pd.DataFrame(data=population, columns=['population'])
```

DataFrame зі списку словників:
```python
data = [{'a': i, 'b': i**2} for i in range(3)]

print(pd.DataFrame(data))
```

Відсутні значення заповнюються NaN (not a number)
```python
print(pd.DataFrame([{'a': 1, 'b': 2}, {'b': 3, 'c': 4}]))
```

DataFrame з двовимірного масиву NumPy:
```python
pd.DataFrame(np.random.rand(3, 2), columns=['foo', 'bar'], index=['a', 'b', 'c'])
```

DataFrame зі структурованих масивів:
```python
A = np.zeros(3, np.dtype({'names': ('foo', 'bar', 'foo2'),
                          'formats': ('U10', 'i8', 'f8')}))

A
```
