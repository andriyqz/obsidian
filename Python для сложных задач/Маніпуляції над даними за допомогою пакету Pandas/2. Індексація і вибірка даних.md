```python
data = pd.Series([0.25, 0.5, 0.75, 1.0],
                  index=['a', 'b', 'c', 'd'])

print(data['b'])
print('a' in data)
print(data.keys()) # Index(['a', 'b', 'c', 'd'], dtype='object')
print(list(data.items())) # [('a', 0.25), ('b', 0.5), ('c', 0.75), ('d', 1.0)]

data['e'] = 1.25
data
```

### Об'єкт Series як одновимірний масив:
Pandas Series надає можливості індексації не гірші, ніж в ndarray, тільки треба звернути увагу, що ``data['a':'c'])`` включає останній елемент, тоді як неявна індексація ```data[0:2]``` - ні.

```
```python
print(data['a':'c'])
print(data[0:2])
print(data[(data > 0.3) & (data < 0.8)])
print(data[['a', 'e']])
```
