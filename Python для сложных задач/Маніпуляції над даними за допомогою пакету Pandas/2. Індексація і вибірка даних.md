### Об'єкт Series як одновимірний масив:
Pandas Series надає можливості індексації не гірші, ніж в ndarray, тільки треба звернути увагу, що ``data['a':'c'])`` включає останній елемент, тоді як неявна індексація ``data[0:2]`` - ні.

```python
print(data['a':'c'])
print(data[0:2])
print(data[(data > 0.3) & (data < 0.8)])
print(data[['a', 'e']])
```

### Індексатори: loc, iloc та ix  (ix was removed in 1.0.0)
Подібні позначення для зрізів можуть привести до плутаниці. Наприклад, коли об'єкт Series містить у собі числовий індекс, то операція індексації ``data[1]`` буде використовувати явні індекси, а операція зрізу - неявні зрізи у стилі Python.

Для того, щоб використовувати явні зрізи придумали функцію loc, а для неявних iloc, тоді як ix - це те саме, що й звичайні квадратні дужки у стилі Python.

```python
data = pd.Series(['a', 'b', 'c'], index=[1, 3, 5])
print(data[1]) # 'a'
print(data[1:3]) # 3 b 
				 # 5 c
```

```python
print(data.loc[1])
print(data.loc[1:3])
```
```
a
1    a
3    b
dtype: str
```

```python
print(data.iloc[1])
print(data.iloc[1:3])
```
```
b 
3  b 
5  c 
dtype: str
```
