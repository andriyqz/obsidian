### У об'єкті Series можна зберігати навіть двовимірні дані:
```python
index = [('California', 2000), ('California', 2010),
         ('New York', 2000), ('New York', 2010),
         ('Texas', 2000), ('Texas', 2010)]

populations = [1000, 1200,
               1300, 1500,
               1400, 1600]

index = pd.MultiIndex.from_tuples(index)
pop = pd.Series(data=populations, index=index)

```
```python
pop = pop.reindex(index)

pop[:, 2010]
```

```python
pop_df = pop.unstack()
```

### Створення MultiIndex:
```python
print(pd.MultiIndex.from_arrays([['a', 'a', 'a', 'b', 'b', 'b'], [1, 2, 3, 1, 2, 3]]))

print(pd.MultiIndex.from_tuples([('a', 1), ('a', 2), ('b', 1), ('b', 2)]))

print(pd.MultiIndex.from_product([['a', 'b'], [1, 2]]))
```
