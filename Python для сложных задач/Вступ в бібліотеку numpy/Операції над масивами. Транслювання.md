Транслювання (broadcasting) - це набір правил по використанню бінарних універсальних функцій для масивів різної розмірності.

1. Спочатку, вибираємо масив меншої розмірності, до його shape прибавляємо одинички зліва
2. Якщо форма двух масивів в якомось вимірі відрізняється, робимо відповідним вимірам одинакову розмірність
3. Якщо shape однакові - робимо операцію, інакше - throw помилку.

```python
x = np.arange(16).reshape((4, 4))
y = np.arange(4)
z = np.arange(4)[:, np.newaxis]

print(x + 5)
print(x + y)
print(z + y)
```
