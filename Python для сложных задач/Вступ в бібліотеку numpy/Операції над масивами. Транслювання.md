Транслювання (broadcasting) - це набір правил по використанню бінарних універсальних функцій для масивів різної розмірності.

1. Спочатку, вибираємо масив меншої розмірності, до його shape прибавляємо одиничку зліва
2. Робимо одиничку зліва розміром як відповідна розмірність нашого масиву
3. Якщо shape однакові - робимо операцію, інакше - throw помилку.

```python
x = np.arange(16).reshape((4, 4))
y = np.arange(4)
z = np.arange(4)[:, np.newaxis]

print(x + 5)
print(x + y)
print(z + y)
```
