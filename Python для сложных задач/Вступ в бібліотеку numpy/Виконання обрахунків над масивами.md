### Арифметичні операції над масивами
```python
x = np.arange(4)

print(x)
print(x + 5)
print(x - 5)
print(x * 2)
print(x / 2)
print(x // 2)

print(np.add(x, 2))
print(np.subtract(x, 2))
print(np.multiply(x, 2))
print(np.divide(x, 2))
print(np.floor_divide(x, 2))
print(np.power(x, 2))
print(np.mod(x, 2))
print(np.negative(x))
```

### Тригонометричні функції
```python
theta = np.linspace(0, np.pi, 3)

print(theta)
print(np.sin(theta))
print(np.cos(theta))
print(np.tan(theta))

```
