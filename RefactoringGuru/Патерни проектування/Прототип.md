Prototype — це патерн, який дозволяє створювати нові об'єкти шляхом клонування існуючих. Тип копіювання (shallow або deep) визначається реалізацією методу клонування.

Shallow copy - це копіюються **значення полів**, включаючи **адреси вказівників**, але не копіюється пам’ять, на яку вони вказують
Deep copy - це глибоке копіювання, коли ми робимо об'єкти повністю незалежними шляхом враховування того, що деякі атрибути можуть мати в собі посилання на якусь адресу пам'яті, яку ми не можемо просто так взяти у новий об'єкт.

Shallow copy:
```
a.data ───► [10]  
b.data ───┘
```

Deep copy:
```
a.data ───► [10]  
b.data ───► [10]
```
