Prototype — це патерн, який дозволяє створювати нові об'єкти шляхом клонування існуючих. Тип копіювання (shallow або deep) визначається реалізацією методу клонування.

Shallow copy - це копіювання, при якому дублюються значення полів об’єкта, включаючи адреси вказівників. У результаті кілька об’єктів можуть посилатися на одну й ту саму область пам’яті, що створює залежність між ними.

Deep copy - це копіювання, при якому створюється нова пам’ять для всіх динамічних ресурсів, і їхній вміст повністю дублюється. У результаті об’єкти є повністю незалежними.

Shallow copy:
```
a.data ───► [10]  
b.data ───┘
```

Deep copy:
```
a.data ───► [10]  
b.data ───► [10]
```

Shallow copy може призводити до проблем, таких як висячі вказівники, тоді як deep copy усуває ці ризики, проте складніший у реалізації.

```python
import copy

class Prototype:
	def clone(self):
		return copy.copy(self)
		
class Car(Prototype):
	def __init__(self, brand, options):
		self.brand = brand
		self.options = options
		
car1 = Car("BMW", ["AC", "ABS"])
car2 = car1.clone()

car2.options.append("GPS")

print(car1.options) # Wrong: ['AC', 'ABS', 'GPS']  
print(car2.options) # ['AC', 'ABS', 'GPS']

```
